# Asset Monitor

A minimal Next.js webpage with a centered title and dark blue background.

## Setup Instructions

1. Install Node.js (version 18 or higher) from [nodejs.org](https://nodejs.org/)

2. Open your terminal/command prompt and navigate to this project folder

3. Install the required packages:
```bash
npm install
```

4. Run the development server:
```bash
npm run dev
```

5. Open your browser and visit: `http://localhost:3000`

## Project Structure

- `app/page.tsx` - Main page with the centered title
- `app/layout.tsx` - Root layout component
- `app/globals.css` - Global styles with Tailwind CSS
- `package.json` - Project dependencies and scripts

## Features

- Minimal design with dark blue background
- Centered title "My Asset Monitor"
- Responsive layout using Tailwind CSS
- Modern Next.js 14 with App Router
